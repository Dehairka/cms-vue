# CMS Vue -- CURRENTLY IN WORK IN PROGRESS

Create very simply a static website in VueJS

## Quick Setup

```sh
# clone the project
git clone https://github.com/Dehairka/cms-vue.git

# install dependency
npm install

# develop
npm run dev
```
